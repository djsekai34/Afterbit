**Afterbit** no es solo un portfolio; es el eje central de mi aprendizaje durante el **Curso de Especialización en Desarrollo de Videojuegos y Realidad Virtual**.

> **Nota:** *Afterbit* es una identidad de marca y empresa ficticia creada exclusivamente con fines académicos para este curso, sirviendo como paraguas para todos mis proyectos de desarrollo.

Dada mi base como **desarrollador web**, decidí que la mejor forma de documentar mi progreso no era usar herramientas limitadas como Google Sites o WordPress. En su lugar, he aprovechado mis conocimientos técnicos para construir esta plataforma desde cero con **ReactJS**. Este proyecto funciona como un **GDD (Game Design Document) interactivo** y dinámico, permitiéndome fusionar mis dos pasiones: la arquitectura web y la creación de videojuegos.

## 🚀 SETUP DE INSTALACIÓN

### 1. Clonar el repositorio
```bash
git clone https://github.com/djsekai34/Afterbit
cd afterbit
```

### 2. Instalar dependencias
```bash
npm install
```

### 3. Ejecutar en local
```bash
npm run dev
```

Abre `http://localhost:5173` en tu navegador y listo.
