## 🖥️ SOBRE EL PROYECTO

**Afterbit** no es solo un portfolio; es el eje central de mi aprendizaje durante el **Curso de Especialización en Desarrollo de Videojuegos y Realidad Virtual**. 

> **Nota:** *Afterbit* es una identidad de marca y empresa ficticia creada exclusivamente con fines académicos para este curso, sirviendo como paraguas para todos mis proyectos de desarrollo.

Dada mi base como **desarrollador web**, decidí que la mejor forma de documentar mi progreso no era usar herramientas limitadas como Google Sites o WordPress. En su lugar, he aprovechado mis conocimientos técnicos para construir esta plataforma desde cero con **ReactJS**. Este proyecto funciona como un **GDD (Game Design Document) interactivo** y dinámico, permitiéndome fusionar mis dos pasiones: la arquitectura web y la creación de videojuegos.

### 🎯 Puntos Clave:
* **Perfil Híbrido:** Aplicación de estándares de desarrollo web profesional al sector de los videojuegos y la XR.
* **Evolución Continua:** Centralización de todos los prototipos y mecánicas desarrolladas durante el año académico.
* **Aprendizaje React:** Mi primera incursión real en este ecosistema, demostrando adaptabilidad y control de lógica de componentes.
* **Inmersión Visual:** Interfaz de alto contraste con sistema de **Modo Claro / Oscuro**, diseñada para una experiencia de usuario (UX) óptima.
* **GDD Interactivo:** Un espacio dedicado a desglosar las entrañas de mi proyecto final, mucho más allá de una simple página estática.

### 🕹️ Proyectos Destacados:
* **SUPER RODOLFO Y LAS ESFERAS DEL SANTO REINO**: Juego final de especialización (Unity 2D / C#).
* **SUPER RODOLFO STRIKE**: Spinoff de Super Rodolfo juego movil para jugar a los bolos.
* **PIXEL METROID**: Desarrollo core y arquitectura de software en 2D.
* **XR ADVENTURE**: Experiencias inmersivas con Realidad Mixta (Google ARCore).
* **EL MALAGUEÑO VS LOS NASTIQUEROS**: Videojuego shooter 3D.

## 🚀 SETUP DE INSTALACIÓN

Si quieres clonar este sistema en tu entorno local, sigue estos comandos:

### 1. Clonar el repositorio
```bash
git clone https://github.com/djsekai34/Afterbit
cd afterbit
```
## 🚀 SETUP DE INSTALACIÓN

### 1. Instalar dependencias
```bash
npm install
```
### 2. Ejecutar en local
```bash
npm run dev
```

## 📁 ESTRUCTURA DE ARCHIVOS CLAVE

### 🧩 Components
* **`src/components/Barra.jsx`**: Navbar dinámica con cierre automático y gestión de modo oscuro.
* **`src/components/ArribaSiempre.jsx`**: Componente de utilidad que fuerza el scroll al inicio en cada cambio de ruta.

### 📄 Pages
* **`src/pages/`**: Directorio principal que contiene todas las vistas de la aplicación (Inicio, Proyectos, Contacto, etc.).
* **`src/pages/Proyectos.jsx`**: Galería central de Afterbit. Aquí se listan todos los proyectos disponibles con sus descripciones técnicas, tags y accesos directos.
  
### 🎨 Assets
* **`src/Imagenes/`**: Repositorio central de assets visuales, logotipos.
* **`src/Video/`**: Trailer del videojuego Super Rodolfo y las esferas del Santo Reino.
* **`src/audio/`**: Sountrack del videojuego Super Rodolfo y las esferas del Santo Reino.
